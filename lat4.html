Font Style

CSS dapat mengatur style dari text yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style text yaitu sebagai berikut.

  • Text align: digunakam untuk mengatur letak kata atau kalimat
  • color: digunakan untuk memberikan warna pada text
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: mengatur jarak antar kata
